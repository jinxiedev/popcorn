# Air Quality Dashboard

## Setup Environment
**Instruksi setup virtual environment:**
1. Buka terminal/command prompt. Buat dan aktifkan virtual environment terlebih dahulu:
   - Pada Windows: jalankan `python -m venv env` lalu `env\Scripts\activate`
   - Pada Mac/Linux: jalankan `python3 -m venv env` lalu `source env/bin/activate`
2. Jalankan perintah: `pip install -r requirements.txt`

## Run Streamlit App
Jalankan perintah berikut di terminal:
`streamlit run dashboard/dashboard.py`
